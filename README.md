# BROCKSEC

Helpful-CMDS

### Nmap Scanning:
        Basic:
        ``` nmap <IP_Address> ```
        Stealthy
        ``` nmap -sS -F <IP_Address>```


### Dirsearch:
        ```dirsearch --url 'url': lookup directories in a website```

### Dig:
        ```dig -x 'ip_address': Reverse nslookup```
        ```dig 'Domain Name': Get IP address from domain name```
